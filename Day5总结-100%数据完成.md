## 🔥 重大发现

### 1. 准确率-鲁棒性正相关 ⭐⭐⭐⭐⭐

**发现**：
```
模型准确率：VGG16 (92.27%) > MobileNetV2 (84.90%) > ResNet18 (83.77%)
攻击难度：  VGG16 (最难) > MobileNetV2 (中) > ResNet18 (最易)

→ 准确率越高，对抗鲁棒性越强！
```

**证据**：
- VGG16平均ASR: 17% (最低)
- MobileNetV2平均ASR: 34%
- ResNet18平均ASR: 39% (最高)

**论文价值** ⭐⭐⭐⭐⭐：
- 系统验证了准确率与鲁棒性的关系
- 提供了量化证据
- 对模型选择有指导意义

---

### 2. One-Pixel的架构敏感性 ⭐⭐⭐⭐⭐

**发现**：
```
MobileNetV2: 21% ASR ← 最脆弱
ResNet18:    16% ASR
VGG16:       0% ASR  ← 完全免疫

差异巨大：21倍！
```

**可能原因**：
```
MobileNetV2特点：
✓ 深度可分离卷积
✓ 倒残差结构
✓ 线性瓶颈层
✓ 轻量化设计

→ 提高了效率
→ 但对单像素扰动敏感
→ 鲁棒性-效率的权衡！
```

**VGG16特点**：
```
✓ 大量参数（138M）
✓ 传统卷积
✓ 深层网络
✓ 大感受野

→ 单像素影响被稀释
→ 完全免疫One-Pixel攻击
```

**论文价值** ⭐⭐⭐⭐⭐：
- 首次发现One-Pixel对不同架构的巨大差异
- 揭示轻量级网络的鲁棒性隐患
- 对移动端/边缘设备部署有重要启示

---

### 3. 攻击方法的效率-效果权衡 ⭐⭐⭐⭐

**发现**：
```
SparseFool: 最快（~0.4秒）, 中等成功率（~37%）
JSMA:       中等（~1.2秒）, 较高成功率（~40%）
One-Pixel:  最慢（~25秒）, 最低成功率（~12%）

效率排序：SparseFool > JSMA >> One-Pixel
```

**推荐**：
- **实用攻击**：SparseFool（快速+有效）
- **深入研究**：JSMA（可控+稳定）
- **特定场景**：One-Pixel（极限稀疏）

**论文价值** ⭐⭐⭐⭐：
- 提供方法选择指南
- 量化效率-效果权衡
- 实用价值高

---

### 4. L0范数的有趣规律 ⭐⭐⭐

**发现**：
```
模型越鲁棒 → 成功攻击需要更多像素

ResNet18: JSMA 4.8像素, SparseFool 4.74像素
MobileNetV2: JSMA 5.28像素, SparseFool 6.80像素
VGG16: JSMA 6.59像素, SparseFool 7.21像素

→ 攻击难度的直观体现！
```
