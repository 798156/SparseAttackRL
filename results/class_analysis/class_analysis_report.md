## 3. 可能的原因分析

### 3.1 为什么某些类别更难攻击？

基于结果，我们推测以下因素可能影响攻击难度：

**1. 视觉特征复杂度**
- 简单纹理（如船、飞机）可能更容易攻击
- 复杂纹理（如猫、狗）可能更难攻击

**2. 类内变异性**
- 类内差异大的类别（如狗）更难定义统一的攻击策略
- 类内一致性高的类别（如汽车）可能更容易攻击

**3. 类间语义距离**
- 与其他类别语义相近的类别更容易被误分类
- 独特的类别（如青蛙）可能需要更多修改才能跨越决策边界

**4. 训练数据分布**
- 模型在某些类别上训练得更好，导致更强的鲁棒性
- 数据增强可能对某些类别更有效
