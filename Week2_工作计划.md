## 📅 Day 6-7：参数敏感性分析

### **任务：**
分析不同参数对攻击性能的影响

### **实验设计：**
```python
模型：ResNet18（代表性模型）
参数：
  - max_pixels: [5, 10, 15, 20]
  - perturbation_size: [0.1, 0.2, 0.3]
方法：JSMA, SparseFool, Greedy（3个主要方法）
样本数：50/配置

总测试：3方法 × 4参数 × 50样本 = 600个
预计时间：2-3小时
```

### **产出：**
1. 参数敏感性曲线图（ASR vs max_pixels）
2. 最优参数表格
3. 稳定性排名（哪个方法对参数最不敏感）

### **脚本：**
```bash
python analyze_parameter_sensitivity.py
```

---

## 📅 Day 8-9：失败案例分析

### **任务：**
深入分析哪些样本难以攻击，为什么失败

### **分析内容：**
1. **统计分析**
   - 所有方法都失败的样本（hard samples）
   - 部分方法失败的样本（method-specific failure）
   - 失败率vs类别的关系

2. **原因分析**
   - 梯度消失/爆炸
   - 决策边界距离
   - 类别语义相似性

3. **可视化**
   - 失败样本展示
   - 梯度热图
   - 决策边界可视化

### **产出：**
1. 失败案例统计表
2. 失败原因分析报告
3. 失败样本可视化（6-10张图）

### **脚本：**
```bash
python analyze_failure_cases.py
```

---

## 📅 Day 10-11：对抗样本可视化

### **任务：**
创建高质量的对抗样本可视化

### **可视化内容：**
1. **对抗样本对比**（原图 vs 对抗图 vs 差异）
   - 每个方法3-5个成功案例
   - 总共15-25张对比图

2. **像素修改热图**
   - 不同方法的像素选择模式
   - 背景vs前景vs边缘

3. **攻击过程可视化**（可选）
   - 逐步展示像素修改
   - 生成GIF动画

### **产出：**
1. 6-10张高质量对比图（用于论文）
2. 像素位置分布热图
3. 攻击过程GIF（可选）

### **脚本：**
```bash
python visualize_adversarial_examples.py
python create_pixel_heatmap.py
```
